bintrade-rust

This is a simple trading app for Linux for Binance using egui, tokio.


Features:
-Trade live binance assets
-Download financial data for assets from Binance/Yahoo finance
-Historic trading with init start money historic with savefiles
-Simulate trading live


How to use:
Install:
cargo install bintrade-rs

I am new to rust so PRs, suggestions, constructive criticism welcome
